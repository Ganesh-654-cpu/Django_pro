# Django_pro --- (Task_Crud)
# Django CRUD Project

## Overview

This is a Django-based CRUD (Create, Read, Update, Delete) application developed using Python and Django. The project demonstrates how to perform database operations, user authentication, image uploads, and template rendering using Django's MVT architecture.

## Features

* User Registration
* User Login and Logout
* Create Records
* View Records
* Update Records
* Delete Records
* Image Upload Functionality
* Django ORM Integration
* Bootstrap User Interface
* Django Admin Panel

## Technologies Used

* Python 3.x
* Django 6.x
* SQLite Database
* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Git & GitHub

## Installation

### Clone the Repository

```bash
git clone https://github.com/Ganesh-654-cpu/Django_pro.git
```

### Navigate to Project Directory

```bash
cd Django_pro
```

### Install Dependencies

```bash
pip install django
```

### Apply Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### Run the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```

## Admin Access

Create a superuser using:

```bash
python manage.py createsuperuser
```

Then access the Django admin panel at:

```
http://127.0.0.1:8000/admin/
```
